# 🤖 AI Sentiment Analyzer

A web application that analyzes text sentiment using AI.

## 🚀 Features
- Real-time sentiment analysis
- Positive/Negative/Neutral detection
- Confidence scores
- Analysis history
- Visual charts

## 🛠️ Technologies
- Flask (Python)
- Hugging Face Transformers
- HTML/CSS/JavaScript

## 📦 Installation

```bash
pip install -r requirements.txt
python app.py
